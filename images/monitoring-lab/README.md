# Monitoring Lab Images

Place the following images in this directory:

1. `go-app-dashboard.png` - Go App Dashboard showing CPU, Memory, MongoDB Latency, Operations Rate, and Latency Over Time
2. `spring-boot-dashboard.png` - Spring Boot Dashboard showing CPU, Memory, MongoDB Latency, Operations Rate, and JVM Heap Memory
3. `comparison-dashboard.png` - Comparison Dashboard showing side-by-side latency and memory usage between Go and Java

The images should be referenced in the blog post as:
- `/images/monitoring-lab/go-app-dashboard.png`
- `/images/monitoring-lab/spring-boot-dashboard.png`
